Este repositório contém meus estudos, exercícios e anotações sobre o núcleo da linguagem JavaScript.
A parte mais introdutória é do curso Introduction to JavaScript da Test Automation University.


Conteúdos Abordados
1. Lógica e Controle de Fluxo

    Condicionais: Estruturas if, else if e else.
    Condicionais Complexas: Uso de operadores lógicos (&&, ||, !) para verificações robustas.
    Laços de Repetição: Automação de tarefas com for e controle de fluxo com while.

2. Estruturas de Dados e Organização

    Arrays: Manipulação de listas e métodos principais.
    Objects: Criação de estruturas chave-valor para representar entidades.
    JSON: Compreensão do formato de intercâmbio de dados e integração com JavaScript.

3. Modularização e Classes

    Functions: Declaração, parâmetros, retorno e reutilização de código.
    Classes (POO): Introdução à Orientação a Objetos, instanciamento e organização de código profissional.

4. Integração com Web

    HTML e JS: Como conectar a lógica do JavaScript com documentos HTML para criar páginas dinâmicas.


Tecnologias e Ferramentas

    Linguagem: JavaScript 
    Ambiente: Navegador Chrome e VScode
    Dados: JSON (JavaScript Object Notation), JS
    Marcação: HTML5
